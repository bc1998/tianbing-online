医德医风管理软件是一套基于B/S结构的为大型医院行风办定制开发的一套系统，系统对医院内部的科室，医师进行整体的考核评定,对所有数据进行统计，分析，导出报表，并且可以根据不同医院的调查项目制定不同的调查表格和随访记录表格，最终生成不同的调查表格，系统中80%的表单全部让医院自己可以定义，大大提高了整个软件的灵活性。
